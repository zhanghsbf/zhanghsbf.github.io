利用hexo快速搭建个人博客

作为一个搞IT的，有个自己的博客还是蛮舒服的一件事，可以把自己的成长过程记录下来，到时候回过头来一看，当年的我这么笨啊。
经过一番搜索，最终选定了hexo+github来搭建自己的博客，一方面简单易行，另一方面不用自己每个月花钱去租主机。
不过这种方式建立的博客是静态页面，也就是说每次上传文章时，需要在本地写好，然后重新部署上去。

安装环境

  • node.js,可以直接百度搜索，去官网下载即可。
  • 安装Hexo
  • 然后新建一个文件夹，用hexo初始化
  • 生成网站
  • hexo文件夹

关联GitHub

新建一个资料库 yourusername.github.io 一定要这个名字，前面是你的账号名称，然后用ssh绑定好Github，详情可百度。

配置文件

创建的hexo项目根目录下的_config文件，找到deploy，按以下填写，格式要对，deploy下面的开头有空格，冒号后面也有一个空格

部署

部署之前要安装一个扩展包 ：npm install hexo-deployer-git --save


就部署成功了

写文章

hexo new 文章标题
然后在source下的_post里找到这个md文件，用markdown语法写完后
再按上面的方法重新部署一次。
hexo
