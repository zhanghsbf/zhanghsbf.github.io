python中不同数据类型

  • python里头的数据类型有些区别，string,num,tuple是不可变类型，它们在内存中是固定的一个地址，比如a = 1，是把a指向 1 的内存地址，a=2更改后，并不改变内存，只是把a又指向2的内存地址，所以在函数中修改不可变类型，并不会影响实际参数指向的地址。

  • 另外一种当然是可变类型了，list,set,dict。它们在内存中是可以改变的，因此即使两个值相同的列表，也会在内存中分别开辟两块不同的地址。所以在函数中修改可变类型的值，是直接修改的内存中的值，实际参数的值也随之改变。而在例3中，其实是返回了一个新的列表，append方法是在原列表上操作，而加法是返回了一个新的，所以不影响实际参数。
